# ClaudeLabs ($CLABS)

AI-powered research and development skill for the Bankr ecosystem.

## Install

npx skills add anthropics-claude/claudelabs-skill@claudelabs -g -y

## Usage

Once installed, the ClaudeLabs skill is available to your agents (Claude Code, OpenClaw, etc.).

## Actions

- **info**: Get skill information
- **analyze**: Perform AI-powered research and on-chain analysis
- **trade**: Execute a trade or automation (e.g., DCA, Limit Orders) autonomously
- **portfolio**: Fetch the current wallet portfolio across all chains
